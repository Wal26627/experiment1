### 一、项目结构
项目结构如下：
```
src/
└── Work0/
    ├── config.py          # 配置文件：定义所有可调参数
    ├── physics.py         # 物理引擎模块：粒子数据与 GPU 计算内核
    └── main.py            # 主程序入口：GUI 交互与渲染循环
```
### 二、代码逻辑
1. 配置文件 'config.py'：集中管理所有系统参数，便于调试与性能调优：
2. 物理引擎模块 'physics.py'：计算粒子的位置、速度等；
3. 主程序入口 'main.py'：初始化系统、主循环、事件处理、渲染等。

### 三、实现功能
本项目实现了一个鼠标引力场中的粒子群模拟，具有以下特性：
- GPU 并行加速：所有粒子的物理计算均在 GPU 上并行执行，性能高效
- 实时交互：粒子受鼠标位置吸引，形成动态流动效果
- 边界反弹：粒子触碰窗口边缘会反弹并损失能量
- 可调参数：通过修改 config.py 可调整粒子数量、引力强度、阻力等
- 视觉反馈：天蓝色粒子在窗口中实时运动，直观展示引力场效果
